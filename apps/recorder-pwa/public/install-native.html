Install native Android app

v1.0.7+ · arm64 build (~10 MB). Background GPS needs location Always + notifications.

Download APK

Use Samsung Internet or Chrome. If download stalls, use USB (below).

Download RNZ-Row-Recorder.apk

Releases page · Build status (if the button 404s, wait ~5 min after a push, then refresh).

Samsung: download stuck or failed?

  1. Do not use a 40 MB build — old universal APKs fail on many phones. New builds are ~10 MB (arm64 only).
  2. USB from PC (most reliable): copy RNZ-Row-Recorder.apk from the project folder apps/recorder-pwa/public/downloads/ or from GitHub Actions artifact → phone Download folder → tap to install.
  3. Swipe down → tap Download complete, or My Files → Downloads.
  4. Allow Install unknown apps for Chrome or My Files.

After install

  1. Settings → Ingest API URL: https://rowing-app-recorder-pwa.vercel.app/api/ingest
  2. Tap Test upload connection (should say OK).
  3. Battery → Unrestricted; Location → Allow all the time
  4. Set Device ID → Start session

Option 2 — Web app (no install)

Open recorder in browser and add to Home screen. Easier install; GPS may pause when the screen locks.
