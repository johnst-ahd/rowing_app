Install native Android app

For Samsung and other Android phones — install once, no Play Store required.

Download APK (Android)

Download RNZ-Row-Recorder.apk

If the link 404s, the build may still be running — check Releases or Actions.

Option 1 — APK file

  1. Get RNZ-Row-Recorder.apk from your team (GitHub Actions artifact, Google Drive, or USB).
  2. On the phone, open the APK file.
  3. If prompted, allow Install unknown apps for Files or Drive.
  4. Open RNZ Row Recorder — the top bar should say Native app.

Samsung settings (once)

  1. Settings → Apps → RNZ Row Recorder → Battery → Unrestricted
  2. Location → Allow all the time
  3. In the app: set Device ID, then Start session

Option 2 — Web app (no install)

Open recorder in browser and use Chrome → Install app / Add to Home screen. Easier, but GPS may pause when the screen locks.
