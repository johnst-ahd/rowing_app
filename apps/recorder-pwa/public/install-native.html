Android install guide

Download APK

v1.0.7+ · arm64 build (~10 MB). Background GPS needs location Always and notifications.

Use Samsung Internet or Chrome. If download stalls, use USB (below).

Download CrewSight.apk

Releases page · Build status (if the button 404s, wait ~5 min after a push, then refresh).

Install steps

  1. Download the APK above.
  2. Open the file and tap Install.
  3. If blocked, allow installs from your browser in Android settings.
  4. Open CrewSight and grant location (Always), motion, and notifications.
  5. Enter your device ID and start a session when on the water.
For sideloading from a PC, copy the APK to the phone and open it with Files or My Files.

USB install (adb)

Enable USB debugging, connect the phone, then run on your computer:

adb install -r CrewSight.apk
