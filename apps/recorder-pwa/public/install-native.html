Samsung: stuck at “Downloading… 100%”?

The file is usually already on your phone. Chrome does not always open the installer automatically.

  1. Swipe down from the top of the screen (notification shade).
  2. Tap the notification Download complete or RNZ-Row-Recorder.apk.
  3. If nothing appears: open My Files → Downloads → tap RNZ-Row-Recorder.apk.
  4. Or in Chrome: ⋮ menu → Downloads → tap the APK.
  5. Tap Install. If asked, allow Install unknown apps for Chrome or My Files.

After install

  1. Open RNZ Row Recorder — the top bar should say Native app.
  2. Settings → Apps → RNZ Row Recorder → Battery → Unrestricted
  3. Location → Allow all the time
  4. In the app: set Device ID, then Start session

Option 2 — Web app (no install)

Open recorder in browser and use Chrome → Install app / Add to Home screen. Easier, but GPS may pause when the screen locks.
