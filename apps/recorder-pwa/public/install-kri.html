After install

  1. Settings → Ingest API URL: https://rowing-app-recorder-pwa.vercel.app/api/ingest
  2. Tap Test upload connection (should say OK).
  3. Battery → Unrestricted; Location → Allow all the time; Notifications on
  4. Set Device ID → start session with GPS and Accelerometer
